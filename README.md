# DMAP-AI: Drought Monitoring and Analysis Platform – AI

**DMAP-AI** is a web-based drought monitoring and analysis platform developed by **AgriMetSoft LLC**.

The current public Research Version provides free point-based **Standardized Precipitation Index (SPI)** drought analysis using **NASA POWER** by default and **ERA5-Land/CDS** as an advanced option.

## Main features

The current DMAP-AI Research Version supports:

- Point-based SPI drought analysis
- NASA POWER precipitation data
- ERA5-Land/CDS advanced data option
- SPI tables and charts
- Drought severity metrics
- Wavelet diagnostics
- Copula diagnostics
- AI-assisted interpretation
- JSON output
- ZIP export of selected tables, charts, and metadata
- Saved projects and saved runs

## Research Version workflow

The Research Version is designed for researchers, students, consultants, engineers, scientists, instructors, and drought analysts who need a fast online workflow for point-based drought assessment.

A typical workflow includes:

1. Select a point location.
2. Choose the data source and analysis period.
3. Run SPI drought analysis.
4. Review tables, charts, drought severity, wavelet diagnostics, and copula diagnostics.
5. Request AI-assisted interpretation.
6. Export selected outputs or copy the structured JSON.
7. Save the run for later review.

## Scientific methods

DMAP-AI currently documents the following calculation components:

- Precipitation aggregation
- SPI calculation using Gamma distribution
- Empirical SPI fallback for numerical stability
- Classic SPI drought categories
- USDM-style SPI communication labels
- Drought event duration and magnitude
- Wavelet power and global spectrum
- SPI–precipitation coherence
- Gaussian copula diagnostics and lag-1 drought persistence
